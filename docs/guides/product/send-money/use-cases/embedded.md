Embedded Finance
Copy

This product is in a closed beta. Please reach out to your sales or implementation team if you are interested.

This guide is intended for Wise Platform partners who want to offer their customers an embedded cross-border money movement experience.

The basic steps that this guide outlines are:

Getting access to the API.
Onboarding customers by setting up and verifying account profiles for your customers.
Sending money on the profiles you created.
Embedded Finance Postman Collection

We've created a Postman collection to help you test your integrations easily! It includes:

Our sandbox environment (https://api.wise-sandbox.com)
Common API calls for creating users and profiles
Automated tests that copy response data between calls
Example responses for reference

Fork the collection to get updates automatically. This lets you test the complete flow with minimal setup!
